Workshop 9 - Cloud Security

 

 

This week's focus follows on from the lecture on Cloud Security and is on understanding the importance of cloud IAM (Identity Access Management) and securing resources and applications.

In this workshop you will:

i) Use Cloud IAM to assign roles to users;

ii) Use Identity-Aware Proxy (IAP) to restrict access to applications;

iii) Deploying a Cloud Security Scanner (optional).
