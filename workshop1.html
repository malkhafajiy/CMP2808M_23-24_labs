Introduction

Before you begin this workshop, please ensure you have completed the form here with your Google email address, as you will be given your own Google Cloud Platform (GCP) project space to use later in the module. Google Cloud Project spaces for the module can only be setup by the module coordinator.

This workshop is the first of many the module team have created to help you pick up the basics of cloud design and development, then progress through to more cloud infrastructure design and development. These workshops are designed to help you understand fundamental cloud concepts and as such it is vital you complete ALL of them. Therefore please take the time to go through all of the workshop . You can also continue working on the workshop tasks from home as the software requirements are basic. You are expected to go beyond the material and tasks presented in the workshops by attempting more advanced tasks and materials.

The primary goal of this workshop is to ensure you have your  Google Cloud Hand-on labs / Qwiklabs account setup, as well as familiarising yourself with the tools. Qwiklabs is currently undergoing a branding change and will soon be primarily known as Google Cloud Hands-on Labs instead of Qwiklabs.

When you have completed  this workshop you will have  learned the basics to:

iii) Gain some practical experience with Google Cloud Project spaces;

 

1. Qwiklabs

Qwiklabs is a learning platform that has deep integration with the Google Cloud Platform, essentially it provides a sandboxed environment for you to complete cloud labs through a browser. As a student on this module you have been given access to Qwiklabs to carry out many of the mandatory workshop tasks and tutorials as part of this module. Each Qwiklab exercise carries a 'credit' weighting, credits are required to do each Qwiklab task and these vary from lab to lab. The module coordinator has been allocated enough credits for all students on the module to complete all labs that are presented in each workshop.

When you carry out ANY Qwiklab exercise, please always remember that you must use the temporary GCP login details that are listed to the left of each Qwiklab - DO NOT login to Qwiklabs using your own GCP/gmail account details! Your personal GCP project space is only to be used for the independent tasks that are part of later workshops.
